Autenticacion Multifactor

La autenticación multifactor es una de las bases en seguridad, con esta API se facilita la autenticación en sistemas, aplicaciones y plataformas. (Amazon Web Services, s.f.-a)

Los ataques de denegación de servicio son problema constante a lidiar, con esta API se logra ofrecer un monitoreo, así como herramientas contra posibles ataques DDoS.(Kaspersky, s.f.; Amazon Web Services, s.f.-b).
